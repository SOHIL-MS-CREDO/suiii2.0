## Usage:

Run these commands to run SUII-BOMBER

## > For Termux:

#### To use the bomber type the following commands in Termux:
```
pkg install git
```

```
pkg install python
```
```
pkg install python3
```
```
git clone https://github.com/SOHIL-MS-CREDO/suiii2.0.git
```
```
cd suiii2.0
```
```
chmod +x SUII.py
```
```
python SUII.py

```
